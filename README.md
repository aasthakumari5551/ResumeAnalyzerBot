# AI Resume Analyzer Bot 🤖

An AI-powered Resume Analyzer Bot built using Python, Telegram Bot API, and Gemini AI API.

## Features
- Upload Resume PDF
- ATS Score Calculation
- Skill Matching
- Missing Skills Detection
- AI-based Resume Suggestions
- Resume Enhancement
- Generate Resume in DOCX/PDF
- Interactive Telegram Bot

## Technologies Used
- Python
- Telegram Bot API
- Gemini AI API
- PDFPlumber
- python-docx
- ReportLab

## How to Run

### 1. Clone Repository

```bash
git clone YOUR_GITHUB_LINK
```

### 2. Install Requirements

```bash
pip install -r requirements.txt
```

### 3. Create .env File

```env
BOT_TOKEN=your_token
GEMINI_API_KEY=your_key
```

### 4. Run Bot

```bash
python main.py
```

## Project Workflow

1. Upload Resume PDF
2. Paste Job Description
3. ATS Score Generated
4. Missing Skills Detected
5. AI Suggestions Generated
6. Enhanced Resume Created
